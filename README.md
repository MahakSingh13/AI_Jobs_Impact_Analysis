
# AI Impact on Jobs

## 📊 Overview

This project analyzes the impact of Artificial Intelligence (AI) on employee salaries and job status using IBM Cognos Analytics. The report compares salary data before and after AI implementation and highlights how AI has influenced different job roles.

The analysis includes structured tables, calculated metrics, and visual charts to provide clear insights into workforce changes.

---

## 🎯 Objectives

* **Data Analysis:** Examine employee salary data before and after AI implementation.
* **Comparison:** Identify differences in salary trends across job roles.
* **Visualization:** Use charts to represent salary changes and patterns.
* **Insights:** Understand how AI affects job status (Modified / Unchanged).
* **Technical Skills:** Apply IBM Cognos features like tables, calculations, filters, and formatting.

---

## 📋 Report Description

### 🔹 1. Data Table

A detailed table was created to display employee information including salary before and after AI. This helps in organizing and viewing raw data clearly.

---

### 🔹 2. Summary Table (2×2)

A summary table with 2 columns and 2 rows was created:

* First column: Categories (e.g., Average Salary Before AI, After AI)
* Second column: Calculated values

This provides a quick overview of key metrics.

---

### 🔹 3. Calculations

The following calculations were performed:

* Average Salary Before AI
* Average Salary After AI
* Salary Difference

These calculations help in analyzing the impact of AI on salaries.

---

### 🔹 4. Visualizations

#### 📊 Bar Chart

* Used to compare salary before and after AI
* Helps identify increase or decrease in salaries

#### 📈 Line Chart

* Shows trends and variations in salary data
* Helps understand patterns across employees

---

### 🔹 5. Formatting & Highlighting

* Job_Status column was used to identify changes
* Modified records were highlighted for better visualization
* Proper headings and text descriptions were added for clarity

---

## 📌 Key Insights

* AI has significantly impacted salary structures in some job roles
* Certain employees experienced salary increases after AI implementation
* Some roles remained unchanged, indicating limited AI impact
* Job status helps identify affected vs unaffected employees
